# 🦕 Dino Timeline

An interactive dinosaur timeline web app for kids — built with HTML, CSS, and JavaScript.

## Project Vision

A fun, visually rich, kid-friendly web experience that lets children explore the age of dinosaurs through an interactive scrollable timeline. Each dinosaur era will have clickable cards showing the dinosaur's name, time period, diet, size, and a fun fact.

## Planned Features

- 📅 Interactive scrollable geological timeline (Triassic → Jurassic → Cretaceous)
- 🦖 Clickable dinosaur cards with illustrations
- 📖 Fun facts and stats for each dinosaur
- 🎨 Bright, kid-friendly design
- 📱 Mobile responsive layout
- 🔊 Optional sound effects and animations
